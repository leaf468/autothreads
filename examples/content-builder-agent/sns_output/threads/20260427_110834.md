Anthropic은 조사 중이며 현재까지 내부 시스템 침해 증거는 발견되지 않았다고 밝혔습니다. 그러나 Mythos처럼 공격·방어 양면에서 활용 가능한 도구의 제한 배포가 뚫렸다는 사실 자체가 제품·공급망 보안 관리의 취약성을 드러냅니다.

이 사건은 ‘민간·공급망을 통한 모델 접근 통제’의 중요성을 다시 환기합니다. 기업은 모델 접근 권한·서드파티 감사·로그 보존·권한 분리 정책을 강화해야 합니다.

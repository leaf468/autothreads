### 3. arXiv: D^2-Monitor — 디퓨전 기반 생성 모델의 동적 안전성 검사 제안

> **원제:** $D^2$-Monitor: Dynamic Safety Monitoring for Diffusion LLMs via Hesitation-Aware Routing
> **출처:** arXiv — https://arxiv.org/abs/2605.25893
> **발행:** 2026-05-26

요약:
D^2-Monitor는 디퓨전 계열 생성 모델(특히 이미지·오디오 생성 파이프라인)에 대해 실시간으로 생성 경로의 `hesitation`(모델의 불확실성 또는 갈등 신호)을 감지하고, 위험도가 높은 경로를 별도 안전 라우팅으로 전환해 유해·탈법적 출력 생성을 억제하는 동적 안전성 모듈을 제안합니다. 실험 결과는 기존 정적 필터링보다 유해 콘텐츠 차단 정밀도가 향상됨을 보여줍니다.

**💡 핵심 포인트**
- 생성 과정에서의 불확실성 신호를 안전성 제어에 활용
- 정적 필터링 대비 대화형·실시간 차단 성능 개선
- 생성 품질 저하를 최소화하면서 안전성 강화 가능

**🤔 추가 인사이트**
국내 이미지·미디어 생성 서비스는 D^2-Monitor 같은 실시간 안전성 계층을 도입하면 규제 대응과 사용자 신뢰 향상에 유리합니다. 다만 모델별 `hesitation` 정의와 임계값 튜닝, 오탐 최소화를 위한 국내 콘텐츠 기준 로컬라이징 작업이 필요합니다.

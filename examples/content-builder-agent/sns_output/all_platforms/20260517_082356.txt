## 📰 주요 뉴스 (3개)

### 1. SANA-WM — 분 단위 영상 월드 모델의 진일보

> **원제:** SANA-WM, a 2.6B open-source world model for 1-minute 720p video
> **출처:** NVlabs / SANA 프로젝트 페이지 (NVlabs)
> **링크:** https://nvlabs.github.io/Sana/WM/
> **발행:** 2026-05 (프로젝트/데모 갱신)

NVlabs의 SANA 계열에서 공개한 SANA-WM은 ‘분 단위(1분), 720p급’ 동영상을 대상으로 한 월드 모델(world model) 구현을 목표로 합니다. 모델 설계는 비디오 수준의 세계 상태를 효율적으로 표현·예측하기 위한 구조(경량화된 시퀀스 모델, 토큰 압축·추출 전략)를 채택해, 비교적 실사용 가능한 해상도·길이의 비디오 생성·예측을 목표로 합니다. 데모와 코드·데이터(샘플 비디오 포함)가 공개되어 있어 연구자·엔지니어가 직접 실험해볼 수 있습니다.

이 접근은 기존의 단편적 프레임 예측을 넘어서 ‘장면 수준의 추론’(물체 상호작용·장면 전개 예측)을 수행하려는 시도로, 로봇 시뮬레이션·시뮬레이션 기반 학습, 장기 플래닝이 필요한 에이전트 연구에 적용 가능성이 큽니다. NVlabs가 제공하는 데모와 레퍼런스 구현은 개발자들이 로컬 환경이나 클라우드에서 빠르게 실험해볼 수 있도록 구성되어 있습니다.

**💡 핵심 포인트**
- 1분 길이·720p급 비디오를 목표로 한 실용적 월드 모델 공개
- 토큰 압축·효율적 시퀀스 모델로 연산·메모리 비용 절감 시도
- 데모·코드 공개로 연구·산업 적용 실험이 용이

**🤔 추가 인사이트**
한국의 영상·로보틱스 스타트업은 이 기술을 시범 적용해 사용자 행동 예측, 자동 편집, 시뮬레이션 기반 학습 데이터 생성 등에 활용할 수 있습니다. 다만 데이터·저작권·프라이버시(영상 데이터) 이슈와 함께 실제 서비스 적용 시 QoS(프레임레이트·지연) 관리가 관건입니다.

### 2. δ-mem: 대형 언어모델을 위한 효율적 온라인 메모리

> **원제:** $\delta$-mem: Efficient Online Memory for Large Language Models
> **출처:** arXiv (논문)
> **링크:** https://arxiv.org/abs/2605.12357
> **발행:** Submitted 12 May 2026

δ-mem은 고정된(full-attention) 백본을 그대로 두고, 소형의 온라인 상태(연관 메모리 상태 행렬)를 덧셈/델타 규칙으로 갱신하여 과거 정보를 압축·보관하는 구조를 제안합니다. 핵심은 ‘작고 고정된’ 메모리 상태로 백본의 어텐션 계산에 저랭크 보정을 추가해 장기 기억 능력을 확장하는 방식으로, 전체 모델을 재학습하거나 컨텍스트 창을 무한히 확장하는 전통적 접근을 피합니다.

실험에서는 매우 작은(예: 8×8) 온라인 메모리 상태로도 메모리 중심 벤치마크에서 의미 있는 성능 향상을 보여주며, 기존 메모리 기법 대비 효율성과 성능 면에서 경쟁력을 주장합니다. 특히 프롬프트 기반 에이전트·장기 대화·메모리 집약 태스크에서 비용 효과적인 대안으로 주목됩니다.

**💡 핵심 포인트**
- 소형(고정 크기) 온라인 메모리 상태로 백본 어텐션을 보정
- 전체 모델 재학습 없이 장기 기억 성능 향상 가능
- 메모리 집약 벤치마크에서 기존 대비 유의미한 향상 보고

**🤔 추가 인사이트**
한국 기업들이 대화형 서비스(업무 비서·상담 챗봇 등)에 적용할 때, δ-mem은 서비스 비용을 크게 올리지 않으면서 ‘사용자별 장기 상태’(사전 선호·이전 대화 이력)를 유지하는 수단으로 유용할 수 있습니다. 다만 실제 제품 적용 전에는 안정성(메모리 오염), 개인정보 보관 정책(장기 상태에 무엇을 저장할지)에 대한 검토가 필요합니다.

### 3. Self-Distillation Enables Continual Learning — 시연형 연속 학습을 향한 방법론

> **원제:** Self-Distillation Enables Continual Learning
> **출처:** arXiv (논문)
> **링크:** https://arxiv.org/abs/2601.19897
> **발행:** Submitted 27 January 2026

Self-Distillation Fine-Tuning(SDFT)는 모델이 자체 생성한 데모(시연)를 교사 신호로 삼아 온-폴리시(on-policy) 학습을 가능하게 하는 단순한 방법론입니다. 전통적 감독적 미세조정(SFT)이 오프-폴리시 성격을 갖는 것과 달리, SDFT는 데모 조건화 모델을 이용해 모델 스스로 온-폴리시 신호를 만들어내어 새로운 기술 습득 시 기존 능력의 쇠퇴(망각)를 줄이는 것을 목표로 합니다.

논문은 여러 스킬 학습·지식 축적 실험에서 SDFT가 SFT보다 신기술 습득 정확도는 높고, 기존 능력의 보존에서도 우수함을 보였다고 보고합니다. 즉, 연속적으로 태스크를 추가해도 성능 저하 없이 능력을 누적하는 ‘실용적 경로’를 제시합니다.

**💡 핵심 포인트**
- 자체 시연을 교사로 사용하는 Self-Distillation 방식의 미세조정
- 연속 학습(continual learning) 상황에서 망각 감소·신규 스킬 학습 향상
- 온-폴리시 신호를 데모 기반으로 생성 → 보강학습 없이 적용 가능

**🤔 추가 인사이트**
기업용 파인튜닝 파이프라인에서 SDFT는 주기적 모델 업데이트 시 기존 기능 유지를 보장하면서 신규 데이터·스킬을 추가하는 방법으로 유용합니다. 특히 비용·보안 제약 때문에 대규모 리트레이닝이 어려운 환경에서 현장 적용 가능성이 큽니다.

### 3. Self-Distillation Enables Continual Learning — 시연형 연속 학습을 향한 방법론

> **원제:** Self-Distillation Enables Continual Learning
> **출처:** arXiv (논문)
> **링크:** https://arxiv.org/abs/2601.19897
> **발행:** Submitted 27 January 2026

Self-Distillation Fine-Tuning(SDFT)는 모델이 자체 생성한 데모(시연)를 교사 신호로 삼아 온-폴리시(on-policy) 학습을 가능하게 하는 단순한 방법론입니다. 전통적 감독적 미세조정(SFT)이 오프-폴리시 성격을 갖는 것과 달리, SDFT는 데모 조건화 모델을 이용해 모델 스스로 온-폴리시 신호를 만들어내어 새로운 기술 습득 시 기존 능력의 쇠퇴(망각)를 줄이는 것을 목표로 합니다.

논문은 여러 스킬 학습·지식 축적 실험에서 SDFT가 SFT보다 신기술 습득 정확도는 높고, 기존 능력의 보존에서도 우수함을 보였다고 보고합니다. 즉, 연속적으로 태스크를 추가해도 성능 저하 없이 능력을 누적하는 ‘실용적 경로’를 제시합니다.

**💡 핵심 포인트**
- 자체 시연을 교사로 사용하는 Self-Distillation 방식의 미세조정
- 연속 학습(continual learning) 상황에서 망각 감소·신규 스킬 학습 향상
- 온-폴리시 신호를 데모 기반으로 생성 → 보강학습 없이 적용 가능

**🤔 추가 인사이트**
기업용 파인튜닝 파이프라인에서 SDFT는 주기적 모델 업데이트 시 기존 기능 유지를 보장하면서 신규 데이터·스킬을 추가하는 방법으로 유용합니다. 특히 비용·보안 제약 때문에 대규모 리트레이닝이 어려운 환경에서 현장 적용 가능성이 큽니다.

### 1. SANA-WM — 분 단위 영상 월드 모델의 진일보

> **원제:** SANA-WM, a 2.6B open-source world model for 1-minute 720p video
> **출처:** NVlabs / SANA 프로젝트 페이지 (NVlabs)
> **링크:** https://nvlabs.github.io/Sana/WM/
> **발행:** 2026-05 (프로젝트/데모 갱신)

NVlabs의 SANA 계열에서 공개한 SANA-WM은 ‘분 단위(1분), 720p급’ 동영상을 대상으로 한 월드 모델(world model) 구현을 목표로 합니다. 모델 설계는 비디오 수준의 세계 상태를 효율적으로 표현·예측하기 위한 구조(경량화된 시퀀스 모델, 토큰 압축·추출 전략)를 채택해, 비교적 실사용 가능한 해상도·길이의 비디오 생성·예측을 목표로 합니다. 데모와 코드·데이터(샘플 비디오 포함)가 공개되어 있어 연구자·엔지니어가 직접 실험해볼 수 있습니다.

이 접근은 기존의 단편적 프레임 예측을 넘어서 ‘장면 수준의 추론’(물체 상호작용·장면 전개 예측)을 수행하려는 시도로, 로봇 시뮬레이션·시뮬레이션 기반 학습, 장기 플래닝이 필요한 에이전트 연구에 적용 가능성이 큽니다. NVlabs가 제공하는 데모와 레퍼런스 구현은 개발자들이 로컬 환경이나 클라우드에서 빠르게 실험해볼 수 있도록 구성되어 있습니다.

**💡 핵심 포인트**

- 데모·코드 공개로 연구·산업 적용 실험이 용이

### 2. δ-mem: 대형 언어모델을 위한 효율적 온라인 메모리

> **원제:** $\delta$-mem: Efficient Online Memory for Large Language Models
> **출처:** arXiv (논문)
> **링크:** https://arxiv.org/abs/2605.12357
> **발행:** Submitted 12 May 2026

δ-mem은 고정된(full-attention) 백본을 그대로 두고, 소형의 온라인 상태(연관 메모리 상태 행렬)를 덧셈/델타 규칙으로 갱신하여 과거 정보를 압축·보관하는 구조를 제안합니다. 핵심은 ‘작고 고정된’ 메모리 상태로 백본의 어텐션 계산에 저랭크 보정을 추가해 장기 기억 능력을 확장하는 방식으로, 전체 모델을 재학습하거나 컨텍스트 창을 무한히 확장하는 전통적 접근을 피합니다.

실험에서는 매우 작은(예: 8×8) 온라인 메모리 상태로도 메모리 중심 벤치마크에서 의미 있는 성능 향상을 보여주며, 기존 메모리 기법 대비 효율성과 성능 면에서 경쟁력을 주장합니다. 특히 프롬프트 기반 에이전트·장기 대화·메모리 집약 태스크에서 비용 효과적인 대안으로 주목됩니다.

**💡 핵심 포인트**
- 소형(고정 크기) 온라인 메모리 상태로 백본 어텐션을 보정
- 전체 모델 재학습 없이 장기 기억 성능 향상 가능
- 메모리 집약 벤치마크에서 기존 대비 유의미한 향상 보고

**🤔 추가 인사이트**
한국 기업들이 대화형 서비스(업무 비서·상담 챗봇 등)에 적용할 때, δ-mem은 서비스 비용을 크게 올리지 않으면서 ‘사용자별 장기 상태’(사전 선호·이전 대화 이력)를 유지하는 수단으로 유용할 수 있습니다. 다만 실제 제품 적용 전에는 안정성(메모리 오염), 개인정보 보관 정책(장기 상태에 무엇을 저장할지)에 대한 검토가 필요합니다.

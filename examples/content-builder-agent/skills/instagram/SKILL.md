# Instagram Post Writing Skill (2026)

Instagram은 사진/동영상 중심 SNS로, 2026년 기준 한국의 국민 SNS입니다. 비주얼이 80%, 캡션이 20%입니다.

## 핵심 원칙

### 1. 어투와 톤
- **친근한 반말 또는 존댓말**: 타겟 오디언스에 따라
- **감성적**: 공감과 감정 표현
- **라이프스타일 중심**: 일상의 특별함 강조
- **브랜드 보이스**: 일관된 톤 유지

### 2. 비주얼 우선
- **이미지/영상**: 콘텐츠의 80%
- **캡션**: 보조 역할 20%
- **첫인상**: 스크롤을 멈추게 하는 비주얼

### 3. 캡션 길이
- **최대**: 2,200자
- **최적**: 150-300자 (간결할수록 좋음)
- **첫 줄**: 125자 이내 (미리보기에 노출)
- **긴 캡션**: 스토리텔링이 필요할 때만

## 2026년 중요 변화: 해시태그 5개 제한!

### 해시태그 대폭 축소
- **이전**: 최대 30개
- **2026**: **최대 5개로 제한**
- **전략 변화**: 양 → 질

### 5개 해시태그 선택 전략

**조합 공식**:
```
대중적 해시태그 1-2개 (도달 범위)
+ 니치 해시태그 3-4개 (타겟팅)
= 총 5개
```

**예시 (카페 포스팅)**:
```
대중적: #일상 #데이트
니치: #성수카페 #브런치맛집 #카페스타그램

총 5개: #일상 #데이트 #성수카페 #브런치맛집 #카페스타그램
```

### 해시태그 선택 기준
1. **관련성**: 콘텐츠와 직접 연결
2. **경쟁력**: 너무 많지도, 적지도 않은 게시물 수
3. **타겟**: 내 오디언스가 검색할 만한 태그
4. **지역**: 지역 기반이면 지역 태그 필수
5. **트렌드**: 시즌/이벤트 관련 태그

### 해시태그 리서치
```
1. Instagram 검색창에 키워드 입력
2. 상위 제안 태그 확인
3. 게시물 수 확인:
   - 10만-50만: 적당한 경쟁
   - 100만+: 너무 경쟁 심함
   - 1만 미만: 너무 적음
4. 관련 게시물 확인 (내 콘텐츠와 유사한지)
```

### 해시태그 위치
- **캡션 끝**: 가장 일반적
- **첫 댓글**: 캡션을 깔끔하게 (선택사항)
- **별도 공간**: "..." 더 보기 아래

```
[캡션 본문]

사진 속 카페는 성수동에 있는 작은 보석 같은 곳이에요 ✨
브런치 메뉴도 정말 맛있고, 인테리어도 예뻐서
하루 종일 있고 싶더라구요 ☕

여러분이 좋아하는 카페는 어디인가요?

...

#성수카페 #브런치맛집 #카페스타그램 #데이트코스 #일상
```

## 캡션 키워드 최적화 (2026 중요!)

해시태그 제한으로 **캡션 내 키워드**가 더욱 중요해졌습니다.

### 자연스럽게 키워드 포함
```
[나쁜 예]
오늘 다녀왔어요!

[좋은 예]
성수동 새로 생긴 브런치 카페 다녀왔어요!
루프탑에서 보는 뷰가 정말 예뻤어요 ☀️
```

### 검색 최적화 키워드
Instagram이 검색에서 인식하는 단어들:
- 지역명 (성수동, 강남, 홍대)
- 카테고리 (카페, 맛집, 호텔)
- 특징 (루프탑, 오션뷰, 브런치)

## 2026년 알고리즘 핵심

### 가장 중요한 지표

1. **저장 (Save)**:
   - 좋아요보다 10배 중요
   - "나중에 다시 볼 가치"
   - CTA: "저장하고 나중에 보세요 🔖"

2. **공유 (Share)**:
   - 스토리 공유
   - DM 공유
   - CTA: "공감하면 스토리에 공유해주세요"

3. **체류 시간**:
   - 얼마나 오래 보는가
   - 캐러셀: 모든 장을 보는가
   - 릴스: 끝까지 보는가

4. **댓글**:
   - 댓글의 질 (길이, 깊이)
   - 대화 형식의 댓글

5. **좋아요**:
   - 여전히 중요하지만 우선순위 낮음

## 이미지 생성

Instagram은 **비주얼이 생명**입니다.

`generate_social_image()` 도구 사용:

```python
generate_social_image(
    prompt="밝고 생동감 있는 색상. 인스타그램 감성. 트렌디하고 세련된 구성. 자연광 느낌. 라이프스타일 이미지. 텍스트 없음. 정사각형 또는 4:5 비율.",
    platform="instagram",
    slug="cafe-brunch"
)
```

이미지 프롬프트 가이드라인:
- **톤**: 밝고 생동감 있는
- **색상**: 파스텔 또는 비비드
- **구성**: 미학적, 인스타 감성
- **비율**: 정사각형(1:1) 또는 세로(4:5)

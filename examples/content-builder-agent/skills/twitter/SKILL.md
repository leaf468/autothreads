# X/Twitter Post Writing Skill (2026)

X(구 Twitter)는 실시간 대화와 빠른 정보 공유 중심의 마이크로블로깅 플랫폼입니다. 간결함과 타이밍이 생명입니다.

## 핵심 원칙

### 1. 어투와 톤
- **간결하고 직설적**: 핵심만 전달
- **위트와 유머**: 적절한 재치 활용
- **친근함**: 대화하듯 편하게
- **반말/존댓말**: 타겟 오디언스에 따라 선택
  - 개인 브랜드: 반말 OK
  - 기업/전문가: 존댓말 또는 중립적 톤

### 2. 글자 수 제한
- **최대**: 280자 (한글 기준 약 140자)
- **최적**: 100-250자일 때 참여도 최고
- **긴 내용**: 스레드(Thread) 활용

### 3. 문장 구조
- **한 문장 = 한 트윗**: 하나의 핵심 메시지에 집중
- **첫 문장이 전부**: 스크롤을 멈추게 하는 훅 필요
- **줄바꿈 적극 활용**: 가독성 향상

## 트윗 유형

### 1. 단일 트윗 (Single Tweet)
짧고 강력한 메시지 하나

```
AI 에이전트의 진짜 가치는 "자동화"가 아니라 "증강"입니다.

인간을 대체하는 게 아니라,
인간의 능력을 10배로 만드는 것.

#AI #에이전트
```

### 2. 스레드 (Thread)
여러 트윗을 연결해 긴 이야기 전개

**첫 트윗 (훅)**:
```
개발자로 첫 직장 선택할 때 놓쳤던 것들 🧵

3년 전의 저에게 해주고 싶은 조언을 정리했습니다.

1/7
```

**이어지는 트윗들**:
```
2/ 첫 번째: 연봉보다 성장 속도

연봉 10% 더 주는 회사 vs 배울 게 많은 회사

당시엔 연봉을 택했지만, 결과적으로 성장이 더 큰 자산이었습니다.
```

```
3/ 두 번째: 팀 문화 > 기술 스택

최신 기술 쓰는 회사라도 코드 리뷰 없고, 문서화 안 되고, 소통 안 되면 지옥입니다.

반대로 레거시여도 좋은 팀이면 배움이 많습니다.
```

### 3. 리스트형
빠르게 스캔 가능한 형식

```
2026년 마케팅 트렌드 핵심 3가지 🎯

1. AI 개인화가 아닌 '인간화'
2. 숏폼에서 롱폼으로 회귀
3. 커뮤니티 > 팔로워 수

#마케팅트렌드 #2026
```

### 4. 질문/대화형
참여 유도

```
주 4일 근무제 도입한 지 3개월 됐는데요,

생산성: ↑
만족도: ↑
이직률: ↓

여러분 회사는 어떤 근무제 원하시나요?
```

### 5. 공감형
많은 사람이 느낄 수 있는 내용

```
회의 전: "이번엔 꼭 내 의견 말해야지"

회의 중: "..."

회의 후: "아까 말했어야 했는데"

개발자들 공감 ㅋㅋㅋ
```

## 해시태그 사용법

- **개수**: 1-3개가 최적 (최대 5개)
- **위치**:
  - 문장 끝에 자연스럽게
  - 또는 별도 줄에 정리
- **트렌드 활용**: 실시간 트렌딩 해시태그 확인
- **브랜드 해시태그**: 캠페인용 고유 태그

예시:
```
좋은 배치:
AI 에이전트가 바꿀 미래 3가지 👇

[내용...]

#AI #에이전트 #미래기술

---

자연스러운 배치:
#AI에이전트 도입 후 생산성이 40% 올랐습니다.

가장 큰 변화는...
```

## 이모지 사용

- **적극 권장**: 감정 표현과 시각적 구분
- **적정 개수**: 트윗당 1-3개
- **용도**:
  - 🧵 스레드 표시 (첫 트윗에 1/🧵)
  - 👇 아래 내용 주목
  - ✨ 강조
  - 🎯 핵심 포인트
  - 💡 인사이트
  - 📊 데이터/통계
  - ⚡ 빠른/즉각적인
- **대체 용도**: 불릿 포인트 대신 이모지

예시:
```
AI 에이전트 도입 체크리스트 ✅

✅ 명확한 목표 설정
✅ 데이터 품질 확인
✅ 팀 교육 계획
✅ 점진적 확대

하나씩 체크해보세요!
```

## 스레드 작성 가이드

### 스레드가 필요한 경우
- 280자로 표현할 수 없는 내용
- 단계별 설명이 필요한 How-to
- 여러 포인트를 상세히 다뤄야 할 때
- 스토리텔링

### 스레드 구조

```
1/🧵 [강력한 훅 - 호기심 유발]
전체 스레드의 주제와 가치 제안

2/ [맥락 또는 배경]
왜 이 이야기를 하는가

3/ [핵심 내용 1]
첫 번째 주요 포인트

4/ [핵심 내용 2]
두 번째 주요 포인트

5/ [사례 또는 증거]
구체적인 예시나 데이터

6/ [핵심 인사이트]
배운 점이나 중요한 깨달음

7/ [마무리 + CTA]
요약 및 행동 유도
팔로우/리트윗 부탁 (선택)
```

### 스레드 작성 팁
- **각 트윗은 독립적으로도 의미 있게**: 중간부터 읽어도 이해 가능
- **번호 매기기**: 1/7, 2/7 등으로 전체 길이 표시
- **일관된 리듬**: 비슷한 길이와 구조 유지
- **마지막 트윗**: 명확한 마무리와 CTA

## 이미지 생성

트윗에 이미지를 추가하면 참여도가 **크게 증가**합니다.

`generate_social_image()` 도구 사용:

```python
generate_social_image(
    prompt="대담한 색상 조합. 기하학적 패턴과 빛나는 효과. 중앙에 하나의 강력한 비주얼 요소. 텍스트 없음. 모던하고 트렌디한 느낌. 스크롤을 멈추게 하는 임팩트.",
    platform="tweets",
    slug="ai-agents-future"
)
```

이미지 프롬프트 가이드라인:
- **톤**: 대담하고 트렌디
- **색상**: 고대비 색상 (스크롤 중 눈에 띄게)
- **구성**: 심플하고 강렬한 하나의 초점
- **텍스트**: 이미지에 텍스트 넣지 않기
